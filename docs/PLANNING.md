## 3. Riesgos y Mitigaciones

- **Riesgo**: Exceso de animaciones afectando el rendimiento.
- **Mitigación**: Usar transiciones CSS nativas siempre que sea posible y limitar Framer Motion a elementos clave.
- **Riesgo**: Diseño demasiado simple que no parezca "Premium".
- **Mitigación**: Uso de tipografía de gran tamaño, espaciado generoso (whitespace) y micro-animaciones de calidad.
