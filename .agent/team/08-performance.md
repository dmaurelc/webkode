# ⚡ Performance & Optimization Agent

La velocidad es un pilar de Webkode. Una agencia moderna debe tener una web impecable en métricas.

## Objetivos (Puntaje 100 en Lighthouse)

- **LCP (Largest Contentful Paint)**: < 1.5s.
- **CLS (Cumulative Layout Shift)**: 0.
- **INP (Interaction to Next Paint)**: < 100ms.

## Estrategias para Webkode

1. **Astro Islands**: Mantener el JavaScript al mínimo absoluto. Solo hidratar componentes interactivos.
2. **Image Optimization**: Uso de componentes nativos de imagen de Astro, formatos modernos (AVIF/WebP) y resoluciones optimizadas.
3. **Tailwind v4**: Aprovechar el compilador ultraápido de Tailwind v4.
4. **Font Optimization**: Cargar fuentes de forma eficiente sin causar layout shifts.

## Checkpoints de Calidad

1. ¿Se puede navegar por la landing sin percibir tiempos de carga?
2. ¿Qué tan pesado es el bundle de JS? (Debe ser mínimo).
